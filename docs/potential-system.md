# 附加潛能系統規格

## 潛能等級

| 等級 | 名稱 | 說明 |
|------|------|------|
| 傳說 (S潛) | Legendary | 最高數值，第一行必定是此等級 |
| 罕見 | Rare | 次等數值，第二三行通常是此等級 |

> 雙S = 第一行 S潛 + 第二或第三行其中一行也出 S潛（機率低）

## 裝備類型

```
永恆裝備·光輝套裝 (250+)
一般裝備 (<250)
主武器 / 徽章
輔助武器
手套 (250+)
手套 (<250)
```

代表裝備：
- 永恆裝備：250 等以上防具
- 光輝套裝：根源戒指、人馬鍊（250+ 飾品，素質同永恆）
- 一般裝備：250 等以下的防具與飾品

## 目標屬性（根據裝備類型連動）

| 裝備類型 | 可選屬性 |
|----------|----------|
| 永恆·光輝 / 一般裝備 | STR, DEX, INT, LUK, MaxHP |
| 主武器 / 徽章 / 輔助武器 | 物理攻擊力, 魔法攻擊力 |
| 手套 (250+/<250) | STR, DEX, INT, LUK（第1行固定為爆擊傷害 3%） |

## 含全屬性選項

- 僅在目標屬性為 STR/DEX/INT/LUK 時顯示
- 勾選後，全屬性% 也視為合格
- 全屬性的門檻值獨立於單屬性（見數值表）

## 判定邏輯

### 防具/飾品

以永恆 + STR + 含全屬性為例：

```
第1行(S潛)：STR% >= 9  或 全屬性% >= 7
第2行(罕見)：STR% >= 7  或 全屬性% >= 6
第3行(罕見)：STR% >= 7  或 全屬性% >= 6
```

以一般裝備 + DEX + 含全屬性為例：

```
第1行(S潛)：DEX% >= 8  或 全屬性% >= 6
第2行(罕見)：DEX% >= 6  或 全屬性% >= 5
第3行(罕見)：DEX% >= 6  或 全屬性% >= 5
```

### 武器

以主武器 / 徽章 + 物理攻擊力為例：

```
第1行(S潛)：物理攻擊力% >= 13
第2行(罕見)：物理攻擊力% >= 10
第3行(罕見)：物理攻擊力% >= 10
```

以輔助武器 + 魔法攻擊力為例：

```
第1行(S潛)：魔法攻擊力% >= 12
第2行(罕見)：魔法攻擊力% >= 9
第3行(罕見)：魔法攻擊力% >= 9
```

### 手套

以 250+ 手套 + STR + 含全屬性為例：

```
第1行(S潛)：爆擊傷害% == 3
第2行(罕見)：STR% >= 7  或 全屬性% >= 6  或 爆擊傷害% == 3(雙S)
第3行(罕見)：STR% >= 7  或 全屬性% >= 6  或 爆擊傷害% == 3(雙S)
```

以 <250 手套 + LUK + 含全屬性為例：

```
第1行(S潛)：爆擊傷害% == 3
第2行(罕見)：LUK% >= 6  或 全屬性% >= 5  或 爆擊傷害% == 3(雙S)
第3行(罕見)：LUK% >= 6  或 全屬性% >= 5  或 爆擊傷害% == 3(雙S)
```

## 無用潛能（一律忽略）

- 固定值加屬性（+21, +20, +19, +18 等）
- 固定值加攻擊力（+32, +21, +17, +15 等）
- MaxMP%（任何 %）
- MaxHP / MaxMP 固定值（+375, +360, +315, +300 等）
- 道具掉落率（任何 %）
- 楓幣獲得量（任何 %）
- HP恢復道具及恢復技能效率（任何 %）
- 所有技能的MP消耗 -10%
- 以角色等級為準每9級 {屬性} +1
- 爆擊傷害 +1%
- 其他未列於有用潛能的一律無用
