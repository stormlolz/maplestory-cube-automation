# 自動洗方塊

新楓之谷（MapleStory）自動洗方塊工具。透過螢幕截圖 + OCR 辨識潛能文字，自動判斷是否達標，並操控滑鼠執行洗方塊流程。

## 功能特色

- **自動辨識潛能**：使用 PaddleOCR 即時辨識螢幕上的潛能文字
- **自訂目標條件**：支援多行條件組合，靈活設定想要的潛能屬性與數值門檻
- **多種方塊支援**：珍貴方塊、絕對方塊、萌獸方塊（直接洗）及恢復方塊（前後比較）
- **圖形化介面**：PyQt6 打造的直覺操作介面，框選區域、設定條件一目了然
- **快捷鍵控制**：支援 F9 快捷鍵開始/停止，不干擾遊戲操作
- **洗方塊紀錄**：即時顯示每次洗方塊的結果與統計

## 系統需求

- Python 3.12
- Windows 作業系統
- 新楓之谷遊戲視窗需可見（工具透過螢幕截圖運作）

## 安裝

1. 安裝 [uv](https://docs.astral.sh/uv/) 套件管理器

2. 安裝依賴：

```bash
uv sync
```

## 使用方式

### 從原始碼執行

```bash
uv run python main.py
```

### 打包為 exe

```bash
uv run pyinstaller cube_automation.spec
```

### 操作步驟

1. 啟動程式，選擇方塊類型（珍貴、絕對、萌獸、恢復）
2. 點擊「框選潛能區域」，在遊戲畫面上框選潛能文字顯示區域
3. 點擊「框選按鈕區域」，框選方塊使用按鈕的位置
4. 設定目標條件（例如：第1行 攻擊力% >= 12）
5. 按下「開始」或 F9 快捷鍵，工具將自動執行洗方塊流程
6. 達成目標條件時自動停止

## 專案結構

```
app/
├── core/       # 底層工具：截圖、OCR、滑鼠控制、模板匹配、條件判斷、自動化主迴圈
├── cube/       # 方塊策略：SimpleFlow（珍貴/絕對/萌獸）、CompareFlow（恢復方塊）
├── gui/        # PyQt6 介面：主視窗、區域選擇器、條件編輯器、洗方塊紀錄、設定面板
├── models/     # 資料模型：AppConfig、PotentialLine、RollResult
└── paths.py    # 路徑常數
tests/          # pytest 測試
docs/           # 設計文件
```
